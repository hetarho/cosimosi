# spec/principle

Cross-cutting principles every spec document and every code change in cosimosi follows. Each is a single, timeless
rule — they apply regardless of *when* you read them or *how* the project got here.

- [timeless-documents.md](timeless-documents.md) — spec docs are complete, current-state statements; never process or history.
- [code-comments.md](code-comments.md) — comments explain the code in front of the reader, not its history.
