# 15. landing-revamp — 환상적 우주 랜딩 (4 테마 + 플로팅 스위처)

> 랜딩페이지를 "정말 우주에 있는 듯한" 환상적 화면으로 끌어올리고, 4가지 전체-페이지 테마를 플로팅 스위처로 바꿔보게 한다. | 범위: FE(랜딩 전용) | 선행: — | 차단: —

## 목적 (Why)

현재 랜딩은 단일한 2D 평면 인상이다 — 그라디언트 블롭 3개 + 가벼운 별 필드(`NebulaBackground`)뿐이라 "기억의 우주"라는 컨셉의 환상성이 살지 않는다. 또한 히어로의 "더 알아보기"가 Lenis 관성 스크롤과 충돌하는 네이티브 `scrollIntoView`를 써서 목적지를 넘겨버리고, 컨셉 섹션을 건너뛰어 신경과학 섹션으로 점프한다.

이 작업은 (1) 그라디언트·그레인·깊이를 적극 쓴 **환상적 우주 배경 4종**을 만들고, (2) **각 신경과학 시각화(별·시냅스·그래프)를 테마별 시각 언어로 다시 그려** 배경·히어로·글래스 크롬과 함께 통째로 바뀌는 **전체-페이지 테마 4종**을 **플로팅 스위처**로 전환·비교하게 하며(localStorage 지속), (3) 어색했던 memory-linking 그래프를 별-시냅스 메타포로 재설계하고, (4) 스크롤 동작을 **Lenis 기준으로** 바로잡는다. 섹션 구성과 시각화의 *의미*는 유지하되, 그리는 *방식*을 4가지 컨셉으로 끌어올린다.

## 범위

**포함**
- 전체-페이지 테마 4종: `aurora`(오로라 커튼) · `deepfield`(허블 딥필드·그레인) · `liquid`(메시 그라디언트) · `nebula`(볼류메트릭 성운·갓레이).
- 테마마다: 고유 배경 atmosphere 컴포넌트 + 그레인 + 히어로 트리트먼트(그라디언트 텍스트·글로우·액센트) + 글래스 카드 크롬(틴트·블러·보더) 토큰.
- **시각화 시각 언어 4종**: 공유 프리미티브 `VizStar`/`VizSynapse`가 테마별로 별·시냅스를 다르게 그린다. nebula=발광 오브+글로우 곡선, aurora=유기 블롭+리본, liquid=광택 구슬+끈적 곡선, deepfield=회절 성좌점+섬세 선. 모든 신경과학 카드(Engram·Hebbian·TimeWindow·SilentEngram·Reconsolidation·NightlyConsolidation)·ConceptSection·ResonanceSection이 이 프리미티브를 공유한다. 시냅스는 항상 곡선(직선 금지).
- **memory-linking(TimeWindowCard) 재설계**: 어색한 꺾은선 그래프를 버리고, 두 별 + 그 사이 시냅스로 표현 — 시간 간격이 커지면 별이 멀어지고 시냅스가 약해진다.
- 플로팅 테마 스위처(화면 우하단 고정, 4 칩, localStorage 지속). 새로고침해도 마지막 테마 유지.
- 스크롤 수정: 히어로 "더 알아보기" → **`concept` 섹션**(컨셉을 건너뛰지 않음), 스크롤 힌트 화살표도 `concept`. 모두 Lenis `scrollTo`로(없으면 네이티브 폴백).
- 접근성: 모든 모션은 `prefers-reduced-motion`에서 정지/축소. 그레인·배경 애니메이션 포함.

**비목표 (Out of scope)**
- 시각화의 **의미 색상(hue)** 은 바꾸지 않는다 (violet=engram, teal=hebbian, amber=time, coral=silent …) — 색은 개념을 운반하므로 보존하고, *재질·형태·시냅스 스타일*만 테마 시각 언어로 바꾼다.
- 시각화 데이터/인터랙션 로직(슬라이더 수식·hover·회상 버튼·단계 애니메이션)은 그대로. 렌더링 표현만 교체.
- /universe·/dormant 등 다른 라우트의 배경/테마. 이 작업은 랜딩(`/`) 전용. (단, `VizStar`/`VizSynapse`는 추후 우주 캔버스에도 재사용 가능한 형태로 둔다.)
- WebGL/three 신규 도입. 배경은 CSS 그라디언트 + 기존 Canvas 2D 별 필드 + SVG 그레인으로 충분히 환상적으로.
- 카피의 "정직성" 축소. 랜딩은 의도된 최종 비전이므로 문구는 유지(오타·명료성만 교정). MVP에 아직 없는 법칙은 [00.overview](../plan/00.overview.md) v1+ 백로그로 따로 추적.

## 설계 요점

- **테마 = 배경 컴포넌트 + CSS 토큰 세트.** 테마 식별자(`aurora|deepfield|liquid|nebula`)를 랜딩 루트 래퍼의 `data-landing-theme`로 박고, 테마별 토큰은 `index.css`의 `[data-landing-theme="…"]` 블록으로 캐스케이드한다. 크롬(글래스·히어로 그라디언트·액센트)은 토큰으로 재테마, 배경 atmosphere는 테마별 전용 컴포넌트.
- **토큰(랜딩 스코프, `--ld-*`):** `--ld-accent`,`--ld-accent-2`,`--ld-hero-grad`,`--ld-glass-bg`,`--ld-glass-border`,`--ld-glass-blur`,`--ld-glass-shadow`,`--ld-grain-opacity`,`--ld-base`. 전역 `.glass`는 그대로 두고 `[data-landing-theme] .glass`에서만 토큰을 적용(다른 라우트 영향 0).
- **시각화 프리미티브(`ui/viz/`):** `VizStar`(cx·cy·r·color·seed·brightness·active·concept → SVG `<g>`)와 `VizSynapse`(두 점·color·strength·concept → 휘어진 곡선)가 컨셉별 분기로 렌더한다. 어떤 svg viewBox 안에든 박힌다. `synapseCurve`가 2차 베지어 곡선 path를 만든다(직선 제거). 카드는 `useLandingTheme`로 concept를 읽어 내려준다. SVG 그라디언트 id는 `useId`로 인스턴스마다 격리(필터 미사용 → 충돌 없음).
- **상태:** `pages/landing/model/theme.ts` — `zustand` + `persist`(localStorage key `cosimosi.landing.theme`) 단일 스토어. 페이지 로컬(웹 전용)이라 DOM 의존 허용. 타입·테마 메타(이름·태그라인·칩 색)도 여기.
- **배경 빌딩블록:** `GrainOverlay`(SVG `feTurbulence` 데이터URI, `mix-blend-mode: soft-light`, opacity 토큰, 모션시 미세 지터), 강화된 `StarFieldCanvas`(밀도·시차 옵션 추가, 하위호환). 테마별 atmosphere는 이 위에 그라디언트 레이어를 쌓는다.
- **스크롤:** `pages/landing/lib/scroll.ts`의 `useScrollToSection()` — `useLenis()` 인스턴스가 있으면 `lenis.scrollTo('#id',{offset})`, 없으면(=reduced-motion 폴백 트리) `getElementById().scrollIntoView()`.

## 수용 기준 (EARS/GWT)

- 1.1 WHEN 랜딩을 처음 열면 THE SYSTEM SHALL 기본 테마(`nebula`)로 환상적 배경을 렌더하고 우하단에 4칩 스위처를 띄운다.
- 1.2 WHEN 스위처에서 다른 칩을 누르면 THE SYSTEM SHALL 배경·히어로·글래스 크롬을 그 테마로 즉시 전환한다.
- 1.3 WHEN 테마를 바꾸고 새로고침하면 THE SYSTEM SHALL 마지막으로 고른 테마를 복원한다(localStorage).
- 1.4 WHEN 히어로 "더 알아보기"를 누르면 THE SYSTEM SHALL `concept` 섹션 상단으로 부드럽게(Lenis) 스크롤하고, 그 지점을 넘기지 않는다.
- 1.5 WHEN `prefers-reduced-motion: reduce`이면 THE SYSTEM SHALL 배경·그레인·히어로의 무한 애니메이션을 정지하고 정적 프레임만 보인다.
- 1.6 THE SYSTEM SHALL 시각화의 의미 색상(hue)을 테마와 무관하게 보존하되, 별·시냅스의 재질·형태는 테마 시각 언어로 바꾼다.
- 1.7 WHEN 테마를 바꾸면 THE SYSTEM SHALL 모든 신경과학 카드의 별·시냅스를 그 컨셉의 시각 언어로 다시 그린다(슬라이더·hover 등 인터랙션은 유지).
- 1.8 THE SYSTEM SHALL 시냅스를 직선이 아닌 곡선으로 그린다.
