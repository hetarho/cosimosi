- [x] A2 좌측 꾸미기 패널 별 섹션에 **`감정별 별 커스텀하기` 텍스트 버튼**이 있고, 누르면 감정별 별 스튜디오 페이지(신규 라우트)로 이동한다.

- [x] A5 **추상화 단계 슬라이더(0~4)** 하나가 13개 미니 캔버스 전부에 동시 적용되어, 끌면 모든 감정 별이 그 단계의 실루엣으로 바뀐다. 이 슬라이더는 **미리보기 전용**이며 저장·서버 값에 영향을 주지 않는다.
- [x] A6 페이지의 **단일 저장 동작**이 색(emotion_colors full-set)과 형태 오버라이드(emotion_forms 부분 업서트 + 미소유 룩 별가루 구매)를 함께 커밋한다 — 기존 `saveEmotionColors`·`pushSettings`·`purchaseItem` 규약 재사용. 잔액 부족이면 저장이 막히고 부분 적용이 없다.
- [x] A7 **마이페이지의 `감정 색` 섹션이 제거**되고(스튜디오로 이전), 최초 온보딩 게이트 `/emotion-colors`는 색만 다루는 표면으로 그대로 동작한다.
- [x] A8 **proto·DB·BE settings 서비스·저장 규약은 이 변경으로 안 바뀐다**(회귀 경계 — FE 편집 표면 재배치만). 우주 메인 렌더·공유 우주(35/37)·방문·근접 탐험(49) 회귀 없음.
