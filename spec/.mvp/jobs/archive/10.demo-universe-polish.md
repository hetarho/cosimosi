- [x] A4 회상 패널이 열려 있거나 일기/잠든 별 목록 오버레이가 펼쳐진 상태에서는 기억 실험실 컨트롤러가 자동으로 닫히거나 숨겨져 서로 덮지 않는다. 겹쳐보기 화면에서도 단일 우주용 체험 HUD가 함께 뜨지 않는다.
- [x] A5 감정 별 또는 다감정 하루를 띄우면 새 별과 새 시냅스가 우주에 즉시 반영되고, 별·연결이 생겼음을 시각적으로 짧게 강조한다. 별도 설명/요약 카드가 없어도 된다.
- [x] A6 `하루 지나기`는 하루 전진과 `demoConsolidate()` 1회를 적용하고, `한 달 지나기`는 같은 1일 배치 단위를 30회 적용한다.
- [x] A7 시간 이동 전후 체험 우주의 별 개수와 시냅스 개수는 변하지 않는다(새 별 추가 액션 제외). 가지치기는 선 삭제가 아니라 weight/밝기 하향만 한다.
- [x] A8 시간 이동 중에는 사용자가 스프링처럼 요동치는 중간 force-sim 상태를 보지 않고, 조용히 재안정화된 최종 우주를 본다.
- [x] A9 시간 이동 구현은 서버/proto/RPC/DB를 변경하지 않고, 좌표는 여전히 클라이언트 force-sim 내부에서만 산출된다.
