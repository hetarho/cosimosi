- [x] T002 (gen) `proto/cosimosi/v1/memory.proto` **append-only**(헌법5): ① 읽기전용 `rpc PeekMemory(PeekMemoryRequest) returns (PeekMemoryResponse)` + 메시지(`PeekMemoryRequest{memory_id}` / `PeekMemoryResponse{Record record; string fragment_text; string derived_text}`), `option idempotency_level = NO_SIDE_EFFECTS`. ② `RecallMemoryResponse`에 회상 성사/쿨다운 필드 추가(다음 빈 필드 번호) — 예 `bool recalled`·`int64 cooldown_remaining_ms`(쿨다운으로 막혔으면 `recalled=false`+잔여). `pnpm gen`(buf) → 호출부 수정.

### BE — 읽기전용 조회 + 회상 쿨다운 게이트

- [x] T004 회상 쿨다운 게이트: `service.go` `RecallMemory`가 `TouchRecall` **전에** 그 별의 `last_recalled_at`을 읽어(repo에 read 추가 또는 `TouchRecall`이 쿨다운 게이트+성사여부 반환하도록 개정), `now - last < recall_cooldown_ms`면 **부작용 전부 스킵**(touch/reconsolidate/enqueue 안 함)하고 쿨다운 잔여를 응답으로 돌린다. `last_recalled_at`이 없거나(첫 회상) 쿨다운 경과면 기존대로 부작용 수행. handler가 `recalled`/`cooldown_remaining_ms`를 proto에 매핑.
