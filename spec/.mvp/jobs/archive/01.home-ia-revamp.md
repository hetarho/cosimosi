### 1단계 — 셸 프리미티브 (`shared/ui`, 3D 씬 밖 DOM·헌법8, `useCoarsePointer` 분기·헌법4)
- [x] T001 [P] 진입 네비 셸 프리미티브 추가 — coarse=화면 하단 **단일 독(탭바형)**, fine=**우측 진입 레일**. shared는 features를 import 못 하므로(헌법4) *빈 그룹 슬롯*(만들기·탐색·소셜)만 제공하고 페이지가 채운다(`OverlayHost`가 children으로 시트를 받는 패턴과 동일). 플랫폼 분기는 ui 레이어에서만.
- [x] T002 [P] 결과 표면 통일 호스트 — `OverlayHost`의 fine-pointer 분기를 좌측 `SidePanel`에서 **코너 비의존 떠있는 카드**로 바꾼다(coarse=바텀시트 유지). 비차단 규약(열려 있어도 뒤 우주 보임·별 탭/회전 가능, peek 핸들, backdrop pointer-events)·Esc·reduced-motion 유지. `SidePanel.tsx`는 떠있는 카드 프리미티브로 대체/리네이밍하고 `shared/ui/index.ts` 공개 API 갱신.
