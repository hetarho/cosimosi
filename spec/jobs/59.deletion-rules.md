## Acceptance Criteria (from plan/48.deletion-rules)

<!-- The source spec's acceptance criteria. After building, verify each is true in the running code. -->

- [ ] A1 **Neuron classification** is a pure predicate that partitions a removal set's neurons into **orphan** (activated by no live memory outside the removal set) and **shared** (still activated by ≥1 live memory outside the removal set), evaluated as-of removal; it is deterministic and IO-free (data supplied by `memory/pg`) [X1][X4].
- [ ] A2 **Full delete** is diary-scoped: it soft-deletes every `EpisodicMemory` born from the target `Diary` (`episodic_memories.deleted_at = now()`), seals the removal set's **orphan** neurons (`neurons.sealed_at = now()`), and **keeps** its **shared** neurons; no neuron, memory, synapse, or activation row is hard-deleted by this operation [X1][X2][I1].
- [ ] A3 **Full delete weakens each removed memory's synaptic contribution** to shared neurons' synapses via `Depress(strength, deletion.contribution_weaken_amount)` ([18.synapse-plasticity](../plan/18.synapse-plasticity.md)); the synapse's base strength is lowered but the edge is **never deleted**, and shared neurons retain their other bonds [X1][I6].
- [ ] A4 **Letting-go** targets **semantic neurons only** — spatial, entity, emotion, color, and the star (`EpisodicMemory` incl. its emotion columns + seed) are untouched, and the memory is **not** soft-deleted (the star lives); orphan semantic neurons are sealed, shared ones kept + contribution-weakened [X4][I11].
- [ ] A5 After letting-go, the star's position/connections **recompute naturally** with **no** position write — force-sim reads only live neurons/synapses, so removing semantic neurons reshapes layout automatically; the star still renders its emotion color + shape (a content-less silent engram) [X4][I5].
- [ ] A6 **Sealed and soft-deleted facts are excluded from ALL dynamics** — the `GetUniverse` read (confirmed), force-sim layout, read-time decay/effective-value computation, consolidation/replay selection, and nebula color — via one canonical alive-predicate (`deleted_at IS NULL` for memories, `sealed_at IS NULL` for neurons, both endpoints alive for synapses); there is no "invisible-but-still-pulling" ghost [X3].
- [ ] A7 **The system never originates deletion** — no decay, consolidation, force-sim, or read path performs a hard delete; the only hard delete is the user-initiated 30-day sweep owned by [49.release-usecase](../plan/49.release-usecase.md), and it is out of scope here [I1][F3].
- [ ] A8 The rules use `Depress` (LTD) and **never** `Downscale` (SHY) — deletion contribution-weakening is associative/local, distinct from homeostatic sleep-time renormalization [I9].
- [ ] A9 The classification/sealing/exclusion behavior is **server-authoritative** (not FE↔BE golden-parity); the client learns of a removal only by facts disappearing from `GetUniverse`, and its unchanged force-sim/effective-value math runs over the smaller live graph.
- [ ] A10 No new column is added to `neuron_activations` — activation exclusion is transitive via the memory's `deleted_at` and the neuron's `sealed_at` (schema of [16.memory-aggregate-schema](../plan/16.memory-aggregate-schema.md) unchanged) [X1][X4].
- [ ] A11 The only tuning number introduced is `deletion.contribution_weaken_amount`; the classification rule, the sealing writes, and the exclusion predicate are formulas/logic that stay in code (not `values.yaml`).

## Grounding

- Constitution ([I1]–[I11]): [00.overview](../plan/00.overview.md) §3 (_The constitution_). This job is bound by
  **[I1]** (the system never originates deletion — this unit is the codification of the "사용자 본인의 명시적 삭제만
  예외" clause: even the user's removal preserves shared neurons and only withdraws the removed memories' synaptic
  contribution; nothing here hard-deletes), **[I2]** (the `Diary` is never mutated — full delete soft-deletes the
  derived memories only), **[I4][I6]** (the weakening acts on neuron↔neuron synapse edges — no memory↔memory edge

  free consequence), **[I9]** (`Depress`/LTD here, never `Downscale`/SHY), **[I11]** (letting-go touches meaning
  only — emotion/color/star are structurally out of reach).
