## 인수 조건 (acceptance — 기획 관점, changes/03.self-star-lighting에서)
<!-- 출처 스펙의 수용 기준. 구현 후 각 항목이 코드에서 참인지 검증한다. -->
- [ ] A1 별 외양이 셋으로 분리된다 — **반사(중앙 거리→밝기)·자가발광(연결성→글로우)·색(감정)**. 가까운 별은 더 비춰지고, 시냅스 많은 별은 더 스스로 빛나며, 둘은 한 별에 동시에 얹힌다.
- [ ] A2 crystal(deepfield)·ember 면 컷/엣지가 반사 N·L로 또렷이 드러난다(오늘보다 입체적). 한 면이 ndv·N·L 두 큐로 이중 음영되지 않는다(크로스페이드).
- [ ] A3 **반사 = 위치 광학**: reflection은 spec 38 위치(activation 최근성)를 읽을 뿐 radius 식을 안 바꾼다. W_INT 0.3 유지, born center/drift/recall(spec 38 수용 3·4·5) 무회귀.
- [ ] A4 **자가발광 = 의미**: self-glow 세기가 시냅스 degree(+weight)에 비례하고, `λ_glow = λ_base·R_conn·R_recent·R_emo`로 감쇠한다(연결·관련성·감정지속이 글로우를 늦춤). 가지친 별도 degree 보존이라 글로우 잔존.
- [ ] A5 **헌법2**: 연결성 0 + 먼(반사≈0) 별도 self-glow A_MIN 잔광으로 렌더+클릭 가능. 코드에 새 flat floor 없고 바닥은 decay.a_min 단일. A_MIN은 비-bloom 경로.
- [ ] A6 **감정 이중 배선**: 감정이 색(정체성)으로 표현되면서, R_emo가 self-glow 감쇠 저항으로 남아 "감정 강한 기억이 더 오래 빛난다"가 유지된다(셋 중 어느 채널에서도 R_emo가 증발하지 않음).
- [ ] A7 **relevance 보존**: "요즘 토픽 근접" 별이 self-glow에서 더 천천히 어두워진다. 서버 relevance(proto field 13·GetUniverse) 경로가 살아있다(헌법3 안전).
- [ ] A8 **dormant 무변**: `raw activation ≤ 2·A_MIN`로 판정(연결성 무관). 회상하면 activation↑로 중앙·반사↑. 서버 ListDormant cutoff 동일.
- [ ] A9 **focus**: 별 선택 시 비-포커스 별의 (반사+자가발광) 합이 거리·연결성과 무관하게 균일하게 FOCUS_DIM(0.12)로 낮아진다 — 어느 채널도 디밍을 우회하지 않는다.
- [ ] A10 **헌법8**: 단일 InstancedMesh 유지. 반사는 buildStarBody emissiveNode TSL 그래프에서 self-position uniform으로 계산(진짜 PointLight 객체 없음). per-instance recency·연결성·focus가 곱해진다. WebGL2/저사양 litMix=0 분기로 광 연산 생략.
- [ ] A11 **헌법4**: 연결성→glow·recency 스칼라는 entities/memory/model 순수(three-free). 조명/falloff는 entities/star/ui. steiger 통과.
- [ ] A12 **배경 씬**: 랜딩·사인인·초대가 우상단 평행광(positional=0)+litMix=1로 crystal 면/엣지가 드러나고 emissive·halo는 유지된다. 위치 고정 점광·fake key-light는 굽지 않는다.
- [ ] A13 **bloom 분리**: self-glow는 bloom, 반사는 lit_albedo_gain cap으로 사실상 안 번짐. 우주 emissive 톤 조정 시 post-bloom 글로우 무회귀.
- [ ] A14 **SSOT·이론 정련**: 새 스칼라는 전부 `values.yaml`(star_lighting·self_glow + star_render.burst_fade_gain), 코드는 import만. neuroscience §4.2/concept "밝기=의미"가 "self-glow(의미)+reflection(최근)"으로, concept↔§6.1 각도-닻 모순이 "각도=안정 닻"으로 갱신. build·lint·typecheck·steiger·우주 코어 루프 무회귀.
