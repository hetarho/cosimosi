## Acceptance Criteria (from plan/31.time-acceleration-ui)

<!-- The source spec's acceptance criteria. After building, verify each is true in the running code. -->

- [x] A1 A persistent **"우주의 시간" HUD** on the universe page shows the current universe time (= the last diary date) read
      from `GetUniverse.universe_time` via `entities/universe-clock`; when the clock is nil (empty universe) it shows an
      empty-universe affordance, not a date ([T6]).
- [x] A2 `entities/universe-clock` is a **pure** mirror (no three) with a `model` (type + store) and an `api` that maps
      `GetUniverse.universe_time`; it is shared verbatim web + mobile.
- [x] A3 On a **launch that advances the clock**, an **acceleration choreography** plays over `[previous_universe_time,
universe_time]` (from the `LaunchStars` response) and the launched star is revealed **after** it (accelerate → then
      the star appears); a **past-dated** launch (empty interval) plays **no** acceleration ([T2] case 1).
- [x] A4 The acceleration is a **neutral time-passing transition with a reserved slot** for forgetting-dimming ([F]) and
      gist-rising / replay ([C]/[E], [V8][C8]); it exposes the interval + a choreography seam those epics fill without
      re-plumbing, and never blocks or delays the optimistic data path.
- [x] A5 A reusable **sync-consent modal** states the consequence in i18n copy with **예 / 아니오** ([T2] case 2 / [R1a]); it
      returns a decision only (it calls no backend). Its **live trigger (회고하기) is Epic C** (`recall-flow-ui` composes
      it); **예** ⇒ proceed (recall syncs + the acceleration plays), **아니오** ⇒ cancel with the clock unmoved.
- [x] A6 The overlay's **control-state** is one XState machine (`idle → confirming → accelerating → idle`); the clock value +
      advance interval live in **Zustand/Query**, not machine context; per-frame scene effects read via `getSnapshot()`,
      never React state (§3.2).
- [x] A7 The overlay exposes **no** rewind, meaning-layer, or placement control — it only shows time, plays a forward-only
      acceleration, and consents to a sync ([I10][I11]); recall-sync/acceleration never mutates a `Diary` ([I2]).
- [x] A8 **All** user-visible copy resolves through the i18n seam (no raw strings; `lint:raw-strings` passes), in a literary /
      restrained Korean voice.
- [x] A9 **web + mobile parity:** the same widget + three features + the entity mirror run on both apps, sharing `model`/`api`
      verbatim; `*.native.tsx` siblings exist **only** for the overlay / modal / HUD primitives (§3.5).
