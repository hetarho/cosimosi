## Review

- [x] Code review applied (manual deep pass; `/code-review` skill is user-invocation-only in this session) +
      Codex (`gpt-5.5`, background) merged. Applied: palette persists are now serialized module-wide in
      [job 62]'s `change-palette` api (a pending-write queue) — two selections racing across a settings
      remount could previously resolve out of order and leave the SERVER on the older palette while the
      client showed the newer one (wrong color on next boot); a component-local saving flag cannot guard
      this (it dies with the screen), so the ordering guarantee lives at the api, with a regression test.
      Rejected with reason: Codex "setSigningOut(false) in `finally` updates an unmounted hook after a
      successful sign-out" — React 18+ makes setState on an unmounted component a safe no-op (the
      mounted-ref guard is the deprecated pattern); the reset exists for the failure path and is
      documented in the api.
