- [ ] A1 같은 form이라도 seed가 다르면 **실루엣(지오메트리)이 눈에 띄게 다르다**(회전만 다른 게 아님).
- [ ] A2 `abstraction_stage`가 오르면 형태가 단계적으로 더 단순/추상화(1~4 단조).
