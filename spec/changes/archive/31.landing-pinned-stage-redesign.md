# 31. 변경: 랜딩 전면 재설계 — 상단 고정 무대 + 스크롤 콘텐츠 인터랙션

> [plan/15.landing-revamp](../plan/15.landing-revamp.md)의 이미 구현된 동작을 바꾸는 **변경 제안**(WHAT 델타).
> 랜딩의 **인터랙션 모델 자체를 갈아엎는다**: 배경이 고정된 채 카드마다 인터랙션이 흩어져 있던 구조를,
> **상단에 고정된 "무대(stage)" 1개 + 그 아래로 끝없이 스크롤되는 콘텐츠**로 바꾼다. 콘텐츠 영역의 버튼·일기 UI가
> 항상 보이는 무대를 조작해 이론을 시연한다. 동시에 장(章) 구성을 **실제 뇌과학 아크**(부호화→연합→재공고화→
> 망각→요즘의 나→야간 공고화)에 맞게 재구성한다.

## 현재 (as-is)
<!-- plan/15.landing-revamp 중 이 변경이 건드리는 현재 동작. 출발점. -->

- **배경 고정 + 카드별 자족 인터랙션.** `LandingBackground`가 `fixed inset-0`으로 깔리고, 스크롤하면 그 위로
  히어로 → 8개 `JourneyAct`가 지나간다. 각 장의 이론 카드(`EngramCard`·`HebbianCard`·`TimeWindowCard`·
  `ReconsolidationCard`·`SilentEngramCard`·`NightlyConsolidationCard`·`FragmentationCard`·`AmbientMoodCard`·
  `PresentSelfCard`·`ConceptSection`·`ResonanceSection`)가 **자기 안에** 버튼·슬라이더·hover·토글 인터랙션을
  품는다 — 인터랙션의 결과도 그 카드 안에서만 보인다(스크롤하면 화면 밖으로 사라짐).
- **히어로 3D 엠블럼은 히어로 섹션에만.** `HeroSection`의 `ThemedStar`(3D)는 첫 화면에서만 보이고 스크롤하면 사라진다.
  나머지 카드는 2D `VizStar`/`VizSynapse`로 각자 그린다.
- **장 순서(8장)**: `concept`(I) → `engram`(II) → `fragmentation`(III) → `connect`(IV: 헵+시간창) →
  `reconsolidation`(V) → `silent`(VI) → `present`(VII: ambient+요즘의 나) → `nightly`(VIII) → `resonance`(IX) →
  `CtaFooterSection`. `engram`과 `fragmentation`이 별도 장이다.
- **카피·정확도 표류.** 일부 문구·시연이 실제 구현(neuroscience.md의 라이브 모델)보다 초기 계획에 가깝다. 특히
  재공고화 카드는 밝기·hue 드리프트만 보이고 **형태 변화**(추상화 단계)는 안 보이며, 야간 공고화도 4단계를
  카드 안 작은 애니메이션으로만 압축한다.
- 테마/오브제 스위처(`AppearanceSwitcher` FAB)와 테마별 배경(`Vast/Lively/Calm`)·`data-landing-theme`·`--ld-*`
  테마 override가 랜딩 색·분위기를 구동한다. (이 변경에서는 **건드리지 않는다** — §비목표.)

## 목표 (to-be)
<!-- 변경 후 참이어야 하는 동작·계약. 구체적으로. -->

### 1. 상단 고정 무대(stage) + 스크롤 콘텐츠 (인터랙션 모델 교체)
- 히어로에서 **별 심볼이 크게 중앙**에 있다가, 스크롤하면 **위로 밀려 올라가 화면 상단에 고정**되며 **화면 높이의
  약 30%를 차지하는 투명 인터랙션 영역(무대)**으로 축소·전환된다. 무대는 영역(레이아웃 높이)만 차지하고 배경은 투명
  하다 — 그 아래 콘텐츠가 비쳐 보인다.
- 무대가 고정된 뒤, **그 아래 콘텐츠(장별 카피 + 인터랙션 버튼/일기 UI)는 계속 위로 스크롤**된다. 인터랙션을
  일으키는 버튼·컨트롤은 **콘텐츠 영역**에 있지만, 그 **결과는 항상 상단 고정 무대에서** 펼쳐진다(스크롤해도 무대는
  안 사라지므로 인터랙션이 화면 밖으로 밀려나지 않는다).
- 스크롤 위치(=현재 보고 있는 장)에 따라 무대가 어떤 이론을 그릴 준비를 하는지가 바뀐다.

### 2. 기존 랜딩 인터랙션 전면 폐기 → 무대 구동 인터랙션으로 재설계
- 카드 안에 흩어져 있던 슬라이더·버튼·hover·토글 데모를 **전부 제거**하고, 아래 **장별 무대 인터랙션**으로 다시 짠다.
  콘텐츠 영역에는 카피 + 트리거(버튼/일기 UI)만 남고, 시각 시연은 모두 상단 무대에서 일어난다.

### 3. 장별 인터랙션 재설계 (전체 뇌과학 아크)
스크롤 순서대로, 각 장은 **콘텐츠 트리거 → 상단 무대 변화**로 작동한다. 흐름은 [neuroscience.md](../tech/neuroscience.md)의
부호화→연합→재공고화→망각→요즘의 나→공고화 아크를 따른다.

| # | 장(가칭) | 뇌과학 | 콘텐츠 트리거 | 상단 무대에서 일어나는 일 |
|---|---|---|---|---|
| 0 | 히어로 | 큰 그림 (§0) | (없음) | 큰 별 → 스크롤 시 상단 무대로 축소·고정. 진입 CTA(체험 우주/로그인). |
| 1 | 뇌가 곧 우주예요 | 엔그램 (§1.1) | 스크롤 진입 | 무대에 **별 하나가 추가되며 뉴런(시냅스)으로 자연스럽게 연결**되는 애니메이션. |
| 2 | 일기를 별로 나눠 우주에 띄워요 | 부호화: 엔그램+**사건 분할 통합** (§1.1–1.2) | 실제 우리 작성 UI와 **동일한 일기**(작성돼 있음) + "별 나누기" → "별 띄우기" 버튼 | 일기가 **N개 조각 별로 분할**되고 무대로 **떠오른다**(일내 결속 시냅스로 가까이 모임). |
| 3 | 함께 떠올린 기억은 단단해져요 | 연합: 헵 LTP/LTD + 시간 창 (§2.1–2.2) | "함께 떠올리기" / "시간 간격" 버튼 | 무대 두 별 사이 **시냅스가 강화**(`+CO_RECALL_DELTA=0.05`, 상한 1.0)되고, 같은 날이면 시간 보너스(+0.3)가 더 굵게. |
| 4 | 떠올릴 때마다 다시 빚어져요 | 재공고화 (§3.1) | "다시 떠올리기" 버튼 | 무대 별이 **밝기±·hue 드리프트**되고 **형태도 변한다**(추상화 단계 — change 29/job 45 의존). 원본 문장은 그대로(불변) 병치. |
| 5 | 잊어도 사라지지 않아요 | 망각·침묵 엔그램 (§4.1–4.2) | "시간 흐르기" → 별이 어두워짐 / "위에 어두운 별을 클릭해보세요" | 시간이 흐르면 무대 별이 **밝기 바닥(`A_MIN=0.05`)까지 어두워지되 사라지지 않고**, 클릭(재점화)하면 다시 밝아진다. |
| 6 | 지금의 내가 우주를 물들여요 | 요즘의 나: ambient + 경쟁적 할당 (§2.3, §5) | 마음(mood) 선택 | **랜딩 페이지 전체 배경색이 그 감정으로 물든다**(무대 안만이 아니라 전역). 새 별이 그 별무리 쪽으로 끌려가 연결. |
| 7 | 밤마다 우주가 정리돼요 | 공고화·수면 (§5.1) | "밤 보내기" 버튼 | 무대에서 **재활성화 → 재분배 → 요지화(형태 단순화·내용 흐려짐) → 가지치기(약한 링크 밝기↓, 삭제 없음)** 단계가 펼쳐진다. **야간 요지화·재공고화·공고화**를 한 무대에서 보인다. |
| 8 | (선택) 공명 | 소셜·분산 엔그램 (§7, 🚧) | 토글 | 두 우주가 같은 사건을 각자의 별로 — 닫는 비전 비트(소셜은 아직 spec만). |

- 위 표의 시연 수치는 **정전 상수를 import**해 표류를 막는다(`A_MIN`·`HALF_LIFE_DAYS` ← entities/memory,
  `CO_RECALL_DELTA` ← features/recall, 시간 보너스 등 — 현 랜딩과 동일 규칙 유지).
- 무대는 2D SVG(`VizStar`/`VizSynapse`) 기반으로 그린다(히어로 진입 엠블럼만 3D `ThemedStar` 재사용 가능).
  시냅스는 항상 곡선(`synapseCurve`)이다.

## 범위 / 비목표 (회귀 경계)
<!-- 무엇을 바꾸고, 무엇은 안 바꾸나. -->

**바꾼다 (전부 `pages/landing/` 웹 전용)**
- 셸 구조: 배경 고정 + 흐르는 카드 → **상단 고정 무대(≈30vh 투명) + 그 아래 스크롤 콘텐츠**. 무대는 히어로 별이
  스크롤에 따라 축소·고정되어 형성. (`LandingPage`·신규 `StageLayer`/무대 훅·스크롤 진행 훅.)
- 카드별 자족 인터랙션 **전면 폐기** → 콘텐츠 트리거(버튼/일기 UI) + 상단 무대 시연으로 재구성. 기존 섹션 컴포넌트
  (`EngramCard`·`FragmentationCard`·`HebbianCard`·`TimeWindowCard`·`ReconsolidationCard`·`SilentEngramCard`·
  `NightlyConsolidationCard`·`AmbientMoodCard`·`PresentSelfCard`·`ConceptSection`·`ResonanceSection`)를 새 장 구성에
  맞게 재작성/통합/삭제.
- 장 구성: 8장 → **위 §3 아크**(엔그램+사건분할 통합, 망각/요즘의 나/야간 공고화를 실제 모델대로). 카피도 함께 정정
  (예: "일기를 쓰는 순간 별 하나가 태어나요" → "일기를 별로 나눠 우주에 띄워요").
- '일기' 트리거 장(#2)은 **실제 작성 UI와 동일한** 일기 입력/표시를 보여준다(우리 `record-memory` UI 시각 톤 차용).
- '요즘의 나' 장(#6)은 무대 안이 아니라 **랜딩 전역 배경색**을 일시적으로 물들인다(해당 장 동안, 벗어나면 복귀).
- policy/ux/landing.md의 §이론 카드 표·인터랙션 모델·진입 설명을 무대 모델로 갱신.

**비목표 (회귀 경계)**
- **헌법 불변** — 밝기만/삭제 없음(§2, 무대 별·링크는 어두워질 뿐 사라지지 않음), 원본 불변(§1, 재공고화 장에서 원본
  문장 병치), 의미 색 보존(§테마/형태 바꿔도 mood hue 불변), 시냅스 곡선 전용(직선 금지), 3D 씬 안 DOM 금지.
- **entities/theory 데모 보존** — `EngramDemo`·`HebbianDemo`·`TimeWindowDemo`·`SilentEngramDemo`는 **체험 우주의
  이론 모달**이 계속 쓴다. 이 변경은 랜딩 인터랙션만 폐기하고 theory entity는 안 건드린다.
- **테마/오브제 스위처 제거는 이 변경 비범위** — `AppearanceSwitcher`·`entities/appearance`·테마 시스템은 **별도
  태스크**가 다룬다. 이 변경은 현행 테마 시스템 위에서 동작한다(테마 바꿔도 무대·콘텐츠가 정상 렌더되면 됨).
- **백엔드·도메인 계약 0 변경** — 랜딩은 웹 전용 showcase. proto/DB/서버 무변경. 도메인 수치는 인용만(정전 상수
  import), 새 도메인 노브 없음.
- **데모 진입 CTA 유지** — 히어로·CtaFooter의 "체험 우주 시작하기"(`enterDemoMode()`→`/`), "로그인하기"
  (`exitDemoMode()`+`resetDemo()`→`/sign-in`)는 그대로.
- **재공고화 형태 변화는 change 29/job 45에 의존** — 그 작업 전에는 #4 무대가 밝기·색만 변형하고 형태 변형은
  job 45 완료 후 자동 반영(같은 `abstraction_stage` 형태 매핑 소비). 이 변경이 job 45를 기다릴 필요는 없다(형태는
  나중에 켜지는 seam).
- **reduced-motion 폴백 유지** — `prefers-reduced-motion: reduce`이면 무대 전환·뉴런 연결·물들임 애니메이션이
  정적 프레임으로 안착하고 콘솔 에러 없음. Lenis 부재 시 네이티브 스크롤 폴백.
- 공명(#8) 같은 소셜 비전 비트는 🚧로 유지(삭제하지 않음, 단 닫는 비트로 축소 가능).

## 수용 기준 (인수 조건)
<!-- 변경 완료의 판정 기준. job의 "인수 조건" 섹션으로 복사된다. -->
1. `/landing` 진입 시 히어로 별이 크게 보이고, 스크롤하면 **위로 밀려 올라가 화면 상단에 고정**되며 약 30vh의
   **투명 인터랙션 무대**로 축소된다. 그 아래 콘텐츠는 계속 스크롤되고 무대는 화면 밖으로 사라지지 않는다.
2. 기존 카드별 자족 인터랙션(슬라이더·버튼·hover·토글이 카드 안에서만 시연)은 **하나도 남지 않는다**. 모든 인터랙션
   트리거는 콘텐츠 영역 버튼/일기 UI이고, 그 **결과는 상단 고정 무대**에서 펼쳐진다.
3. 장이 §3 표의 **전체 뇌과학 아크**(엔그램 → 일기→별 분할/띄우기[엔그램+사건분할 통합] → 헵·시간 창 →
   재공고화 → 망각 → 요즘의 나 → 야간 공고화)로 재구성되고, 카피가 그에 맞게 정정된다.
4. **뇌=우주** 장: 무대에 별 하나가 추가되며 뉴런(곡선 시냅스)으로 자연스럽게 연결되는 애니메이션이 보인다.
5. **일기→별** 장: 실제 작성 UI와 동일한 일기가 있고, "별 나누기→별 띄우기"로 일기가 N개 조각 별로 분할되어
   무대로 떠오른다(일내 결속으로 가까이 모임).
6. **헵·시간 창** 장: 콘텐츠 버튼으로 무대 두 별 시냅스가 실제 규칙대로 강화(+0.05·상한 1.0)되고 같은 날 보너스가
   반영된다(단조 강화, 정전 상수 import).
7. **재공고화** 장: "다시 떠올리기"로 무대 별이 밝기·색이 변하고(형태 변화는 job 45 완료 시 자동 반영), 원본 문장은
   불변으로 병치된다.
8. **망각** 장: "시간 흐르기"로 무대 별이 `A_MIN=0.05` 바닥까지 어두워지되 사라지지 않고, 어두운 별을 클릭하면 다시
   밝아진다.
9. **요즘의 나** 장: 마음(mood) 선택 시 **랜딩 페이지 전체 배경색**이 그 감정으로 물들고, 장을 벗어나면 복귀한다.
10. **야간 공고화** 장: "밤 보내기"로 무대에서 재활성화→재분배→요지화(형태 단순화·내용 흐려짐)→가지치기(약한 링크
    밝기↓·삭제 없음) 단계가 펼쳐진다.
11. 의미 색(mood hue) 보존·시냅스 곡선 전용·밝기만/삭제 없음·원본 불변 헌법이 무대 시각화에서도 지켜진다.
12. `entities/theory` 데모는 그대로 남아 체험 우주 이론 모달이 정상 동작한다(회귀 없음). 데모 진입 CTA(체험 우주/
    로그인) 정상.
13. `prefers-reduced-motion: reduce`이면 무대 전환·연결·물들임이 정적 프레임으로 안착하고 콘솔 에러가 없다.
14. FE `build`·`lint`·`steiger` 통과, 테마 시스템(현행)·체험 우주·이론 모달 회귀 없음.
