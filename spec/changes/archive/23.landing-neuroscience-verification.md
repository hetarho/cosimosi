# 23. 변경: 랜딩 뇌과학 카드 검증·정정 (jobs 32–38 반영)

> [plan/15.landing-revamp](../plan/15.landing-revamp.md)의 이미 구현된 동작을 바꾸는 **변경 제안**(WHAT 델타).
> jobs 32–38(거리 드리프트·반경 밝기·야간 재설계·고유 형태·재공고화 AI 변형·감정 유사도 링크·간격효과 제거)로
> 모델이 바뀌어, 랜딩 9막 중 일부 카드 문구·시각화가 옛 모델 기준이다. 이를 **검증해 정정하고, 빠진 검증 사실을 보강**한다.

## 현재 (as-is)

랜딩은 9막(concept / engram / fragmentation / connect / reconsolidation / silent / present / nightly / resonance)으로
뇌과학을 설명한다([LandingPage.tsx](../../frontend/src/pages/landing/ui/LandingPage.tsx), 막별 카드는
`pages/landing/ui/section/*Card.tsx`). jobs 32–38이 들어오기 전 모델을 기준으로 쓰여, 다음이 **현재 구현(정전 policy)과 어긋난다**:

- **VI 망각 — [SilentEngramCard](../../frontend/src/pages/landing/ui/section/SilentEngramCard.tsx)**: "오래 떠올리지 않은 기억은
  어두워질 뿐 그 자리에 남는다"로 *제자리 디밍*만 말한다. 시연 카드의 밝기 바닥은 `FLOOR=0.12`이고, 망각을 옛
  `λ_eff`(연결·관련성·감정 변조 감쇠) 그림으로 암시한다.
- **VIII 야간 공고화 — [NightlyConsolidationCard](../../frontend/src/pages/landing/ui/section/NightlyConsolidationCard.tsx)**:
  "흐릿한 디테일은 줄거리만(gist), 약한 연결은 정리"라는 옛 4패스(재안정화·재분배·나이/회상 트리거 요지화·가지치기) 기준.
- **V 재공고화 — [ReconsolidationCard](../../frontend/src/pages/landing/ui/section/ReconsolidationCard.tsx)**: 회상 시 밝기·`form_seed_delta`가
  소폭 흔들리는 것만 보여준다. 별의 *고유 형태/추상화 단계 변화*(spec 53)와 *내용 변형*(spec 54)은 없다.
- **IV 연합 — [HebbianCard](../../frontend/src/pages/landing/ui/section/HebbianCard.tsx)·[TimeWindowCard](../../frontend/src/pages/landing/ui/section/TimeWindowCard.tsx)**:
  "함께 떠올리면 굵어지고, 같은 하루의 인연은 또렷"까지만. **감정 유사도가 링크 가중치에 들어간다는 사실(job 37)은 없다.**
- **연결→거리 사실 누락**: 어디에도 "자주 함께 떠올린·연결 많은 별은 *가까이* 머문다(거리=강함)"가 없다. jobs 32·33으로
  밝기·위치가 **하나의 변수(자기-거리/반지름)**로 통합된 게 랜딩의 어떤 막에도 반영돼 있지 않다.

## 목표 (to-be)

랜딩 전반을 현재 구현(아래 정전 policy)에 맞춰 **검증**하고, 어긋난 문구·시연 수치·시각화를 **정정**하며, 빠진 검증 사실을
**보강**한다. 막 구성(IA)·마케팅 톤은 유지하고, 카드 내용/시연 로직만 진실에 맞춘다.

정전 기준(이번에 갱신된 policy):
- [policy/domain/star.md](../policy/domain/star.md) §밝기·감쇠("별 밝기 = 자기-거리"), §요지화(change 20), §시각 규칙(spec 53 형태).
- [policy/domain/synapse.md](../policy/domain/synapse.md)(감정 유사도 링크·간격효과 제거·야간 링크 reweight).
- [policy/domain/memory.md](../policy/domain/memory.md)(재공고화 AI 내용 변형, spec 54).

정정/보강 항목:

1. **VI 망각 → 거리 드리프트 + 반경 밝기**. "어두워질 뿐"을 **"멀어지며 어두워진다"**로 바꾼다 — 별 밝기는 독립 감쇠가
   아니라 **자기로부터의 거리(반지름)의 단조 함수**이고(R_MIN=6 밝음 ↔ R_MAX=80 `A_MIN`), 고립·저강도·일회성 별은
   바깥으로 빠르게 밀려나 어두워지고(연결·고강도·잦은 회상 별의 ~2–3배), **연결·회상·감정은 거리를 늦춰 가깝고 밝게**
   머문다. 시연 카드의 `FLOOR=0.12`는 정전 바닥 **`A_MIN=0.05`**로 통일하고, `λ_eff`·relevance 그림은 제거한다. "삭제 없음·잠든
   별 회상으로 재점화·불변 원본"은 유지(헌법1·2).
2. **VIII 야간 공고화 → change 20 재설계 반영**. (a) **반경 스코프 ≤60**(멀리 표류한 별은 건드리지 않음=망각 존중),
   (b) **시간 링크는 약화·의미 링크는 강화**("최근 함께 언급"은 바래고 "의미가 닿는"은 오래 남음), (c) **spread**(성단이 한곳에
   뭉치지 않게 결정론적 분산), (d) **요지화 = 별의 반지름이 임계를 넘으면 `abstraction_stage`(0~4) 단조 승급 → 형태가 한 단계
   추상화**(나이/회상 트리거 아님), (e) **마지막 연결 보호**(모든 별 degree ≥1, 고립 금지), (f) **re-KNN**(오래되어 끊긴 별을 새로
   닮은 기억과 다시 잇는 재연결). "약한 선은 빛만 낮춤(삭제 없음)·원본 불변·04:00 정리"는 유지.
3. **V 재공고화 → 고유 형태 + 추상화 + 내용 변형**. 회상이 별을 양방향으로 다시 빚는다는 기존 메시지에 더해, 별은 **저마다
   고유한 실루엣**(다축 시드, spec 53)을 갖고 **요지화될수록 형태가 단순해지며**(추상화 단계), 높은 추상화 단계에서는 **AI가
   회상 내용을 옅게 다시 쓴다**(spec 54, 원본 record는 한 글자도 불변 — 헌법1, 변천사 append-only). 시연 카드 형태 변형을
   "단순 시드 흔들림"에서 "고유 형태 + 단계적 추상화"로 갱신.
4. **IV 연합 → 감정 유사도 링크 보강**. "같은 의미라도 **감정이 비슷한 기억끼리 더 굵게** 이어진다"(job 37)를 추가한다 —
   단, 후보 게이트(`cos ≥ 0.75`)는 불변이고 감정은 이미 의미가 닿는 별들 사이의 *가중치만* 미세조정한다("같은 기분이지만 무관한
   기억"이 엮이진 않음). 간격 효과 문구가 어디 있으면 제거(job 38 — 공동 회상 강화는 간격 무관 고정).
5. **연결→거리 사실 보강**. 적절한 막(예: 연합 IV 또는 망각 VI, 혹은 작은 보강 카드)에서 **"자주 함께 떠올린·연결이 많은 별은
   서로 *가까이* 머문다 — 거리가 곧 강함이다"**를 명시한다. 밝기·위치가 하나의 변수(자기-거리)로 통합됐음을 한 문장으로라도 비춘다.
6. **나머지 막 검증**. concept/engram/fragmentation/present/resonance는 현 policy와 대조해 어긋난 수치·문구가 없는지 확인하고
   (예: ambient τ, 흥분성 게인 `g=1+0.3·각성`, 끌림=의미+0.25·흥분성, 소프트 억제 ×0.5), 틀린 것만 고친다. 맞으면 그대로 둔다.

## 범위 / 비목표 (회귀 경계)

**범위**: `pages/landing/**`의 카드 문구·시연(데모) 로직·시각화만. 정정은 위 6항목과 그 검증에서 추가로 발견되는 명백한 오기.

**비목표 (회귀 경계)**:
- 랜딩 9막 IA·스크롤 구조·라우팅·마케팅 카피 톤·CTA·테마 시스템을 **재설계하지 않는다**(검증·정정·소폭 보강만).
- 새 막을 대거 추가하지 않는다(연결→거리는 기존 막 보강 또는 1개 소형 카드 이내).
- 백엔드·proto·RPC·sqlc·DB schema 변경 없음. 랜딩은 순수 FE 비주얼.
- 헌법 불변식(원본 불변·삭제 없음·서버 비좌표·`A_MIN=0.05` 단일 바닥)은 그대로.
- 데모(`/` 체험) 동작은 이 change 범위 밖(plan 47/48 = change 25·26 소관). 랜딩 카드 내부의 *시연 미니 viz*만 정정한다.

## 수용 기준 (인수 조건)

1. VI 망각 카드가 "제자리 디밍"이 아니라 **거리 드리프트 + 반경 밝기**(연결·회상·감정이 거리를 늦춰 가깝고 밝게 머묾, 고립 별은
   바깥으로 빠르게 멀어져 어두워짐)로 설명·시연한다. 카드의 밝기 바닥이 `A_MIN=0.05`이고 `FLOOR=0.12`·`λ_eff`·relevance 표현이 없다.
2. VIII 야간 공고화 카드가 change 20 재설계(반경 스코프 ≤60·시간링크 약화/의미링크 강화·spread·`abstraction_stage` 요지화·마지막 연결
   보호·re-KNN 재연결)를 반영하고, "약한 선은 빛만 낮춤·원본 불변"을 유지한다.
3. V 재공고화 카드가 별의 **고유 형태(다축 시드)** + **추상화 단계로 형태 단순화** + **높은 단계에서 AI 내용 변형(원본 불변·변천사 append)**
   을 보여준다.
4. IV 연합 카드에 **감정 유사도가 링크 가중치를 미세조정**한다는 사실이 있고(후보 게이트 `cos ≥ 0.75` 불변 명시), 간격 효과를 주장하는 문구가 없다.
5. 랜딩 어딘가에서 **"연결 많은·자주 함께 떠올린 별은 가까이 머문다(거리=강함)"**가 명시된다.
6. concept/engram/fragmentation/present/resonance 막의 수치·문구가 현재 policy와 일치한다(틀린 것만 정정, 맞는 것은 불변).
7. 랜딩 9막 IA·라우팅·마케팅 구조·CTA가 그대로다(검증·정정·소폭 보강 외 재설계 없음).
8. 백엔드·proto·RPC·sqlc 변경 없이 FE만으로 구현되고, build·lint·typecheck·steiger를 통과한다. 랜딩 스크롤·테마·기존 인터랙션 회귀 없음.
